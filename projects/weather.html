Weather2Run



Research | Competitor Analysis | User Interviews and Surveys | Personas
Scenarios | Task Analysis | Mood Board | Icon Design | Wireframing
Prototyping | Interaction Design | User Testing

The challenge of this project was to create a weather phone application for a specific user population. I selected runners as weather is often an important consideration that can severely effect how a person will run or even if they will run at all.

The application shows the user the upcoming weather for the day, the upcoming weather for the week, and the user's tracked running performance.
The application stores user run data based on weather conditions. Users can see how different weather (including humidity, wind, temperature) affects their runs.
